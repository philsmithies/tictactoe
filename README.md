To Run:
```
git clone https://github.com/philsmithies/tictactoe.git
```

Install Yarn if you don't have it already:
```
npm install --global yarn
```

Run Yarn to initialize
```
yarn
```

Run Tests
```
yarn test
```

Run Program
```
Open index.html in your browser
```
